

This project simulates an elevator system based on user-defined parameters.

## How to Run

The program requires **four command-line arguments**:<br>
`<numElevators> <numFloors> <numUsers> <simulationTimeInMinutes>`


### Example

To run the simulation with 3 elevators, 10 floors, 50 users, and a simulation time of 7200 units:

```bash
sbt run 3 10 50 7200
